# 世紀爭霸戰：腸道好菌與壞菌的優勢爭奪（遊戲前言／引子）

> 本篇為《益生菌生存戰爭》的世界觀前言，玩家開場先讀，建立「腸道＝城市、酸鹼值＝戰場」的心智圖。

想像一下，人類的腸道就像是一座熱鬧無比的超級大城市，裡面住著上百兆個微小居民，這些居民就是所謂的「細菌」。科學家在定義這座城市裡的「好菌（益生菌）」與「壞菌（致病菌）」時，其實並非以外表來評斷，而是取決於牠們在城市裡「負責什麼工作」以及「製造出什麼產物」。

## 一、決定善惡的關鍵：「副產物」
細菌和人類一樣需要進食，而牠們飽餐一頓後的「排泄物」，正是決定牠們屬於善或惡的關鍵。
- **好菌（例如：優格裡的乳酸菌）：** 最愛膳食纖維與乳糖，消化後排出「有機酸」（乳酸、醋酸），使腸道偏酸性。
- **壞菌（例如：大腸桿菌、產氣莢膜梭菌）：** 特別愛未消化完的蛋白質與壞脂肪，排出「毒素、氨氣、硫化氫」，呈鹼性且惡臭。

## 二、城市環境的「酸鹼值戰爭」
- 好菌製造酸性物質，是為了劃分並鞏固勢力範圍。
- 壞菌的致命弱點：非常怕酸，只能在鹼性環境大快朵頤。
- 好菌大量繁殖 → 滿滿乳酸 → 城市變酸性 → 壞菌被壓縮、枯萎或排出。
- 壞菌佔上風 → 環境轉鹼 → 毒素腐蝕腸壁、引發發炎。

## 三、「好」與「壞」取決於數量與位置
- **平時的壞人：** 壞菌在健康腸道也存在微量，乖乖待角落時甚至能幫忙分解廢棄物。
- **失控的暴民：** 天天餵鹹酥雞、精緻糖 → 壞菌暴增、壓倒好菌 → 化身破壞城市的暴民。
