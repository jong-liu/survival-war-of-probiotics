## 0. 一句話
外派乳酸菌進腸道城市打造酸性防禦網，**平時用機會卡累積好菌優勢**，面對全桌相同的命運考驗時安然度過，**維持酸性優勢最久者勝**。

## 1. 核心設計決策（已鎖定）
| 項目 | 決策 | 對應子問題 |
|---|---|---|
| 事件模型 | 共用命運牌堆、**無圖板**，每回合翻 1 張全桌同時面對 | Q2 同路不同命 |
| 狀態呈現 | 面前**公開牌陣**，🟢🔴 顏色比例一眼看懂 | Q1 視覺簡化 |
| 結算 | **簡單二元**（過門檻→免罰／沒過→固定受罰） | 易教 |
| 「底子越好越抗壓」 | 靠**命運卡免疫門檻分級**保留梯度 | Q3 |
| 計分 | **累積酸性優勢分**（每回合把好菌−壞菌記點累加） | Q4 維持平衡 |
| 護欄 | 🔴不可歸零（最低留 1）；菌叢區上限 10 張 | Q4 動態平衡、非全滅 |

## 2. 名詞
- **菌叢區**：每位玩家面前的公開牌陣，放 🟢好菌卡 與 🔴壞菌卡。
- **酸性優勢值** = （菌叢區好菌數）−（壞菌數）。正 = 偏酸／健康，負 = 偏鹼／危機。
- **免疫門檻**：每張命運卡標示的數字，酸性優勢值 ≥ 門檻才免罰。

## 3. 元件
- 🟢好菌卡、🔴壞菌卡（共用供應區；紙上原型可用綠/紅便利貼）
- 🃏機會卡牌堆（主動保健，抽成手牌）
- ⚡命運卡牌堆（外在衝擊，共用事件）
- 計分軌／計分紙（數位版由系統自動累加）

## 4. 起始設定
- 每位玩家菌叢區：`2🟢 + 1🔴`（好菌略優，但壞菌永遠有底 → 呼應「壞菌平時也在角落」）
- 每位玩家手牌：抽 3 張機會卡
- 命運牌堆洗好，張數 ≈ 8–10（決定遊戲長度＝一段生活週期）

## 5. 回合流程
每回合翻 1 張命運卡，全桌同時經歷：
1. **揭示命運卡**（含它的免疫門檻與懲罰）。
2. **反應階段**：從起始玩家順時鐘，每人**最多打 1 張機會卡**，立即結算效果。
3. **條件檢查（各自二元結算）**：
   - 酸性優勢值 **≥ 該卡免疫門檻** → **免罰**（安全度過）
   - 否則 → **承受該卡懲罰**
4. **記分**：每人把此刻的（好菌−壞菌）記上計分軌，累加為「累積酸性優勢分」。
5. **補牌**：手牌補回 3 張。

命運牌堆抽完 → 遊戲結束，**累積酸性優勢分最高者「腸道最健康」獲勝**。

## 6. 命運卡免疫門檻分級（保留「底子越好越抗壓」）
結算雖是二元，門檻依事件強度不同，平時堆越多好菌越扛得住大事件：

| 命運卡 | 免疫門檻 | 沒過的懲罰 | 連鎖／殘留 |
|---|---|---|---|
| 鹹酥雞宵夜 | +1 | +1🔴 | — |
| 精緻糖手搖 | +1 | +1🔴 | — |
| 考試壓力爆表 | +2 | +1🔴 | 中招留 🔥發炎 |
| 纖維不足便祕 | +2 | +1🔴 | 中招留 🔥發炎 |
| 熬夜追劇 | +2 | −1🟢 | — |
| 暴飲暴食吃到飽 | +2 | +2🔴 | ⛓️ 下回合連鎖「隔天腹瀉」 |
| 隔天腹瀉（連鎖後續） | +1 | +1🔴 且 −1🟢 | — |
| 抗生素療程 | +3 | −2🟢（連好菌一起殺） | 中招留 💥菌相失衡 |
| 流感大流行 | +3 | +2🔴 且 −1🟢 | 中招留 🔥發炎 |
| （好事件）連假早睡＋運動 | 門檻 0 | **只有好菌>壞菌者** +1🟢 | — |
| （好事件）陽光與好心情 | 門檻 0 | **好菌>壞菌者** +1🟢 | ✨ 淨化全員負面狀態 |

**殘留狀態（影響下回合，事件連鎖核心）**：🔥發炎＝下回合檢定 −1；💥菌相失衡＝下回合開始 −1🟢；🛡️清淡調養＝下回合檢定 +1。

## 7. 機會卡（主動保健，每回合最多打 1 張）
| 機會卡 | 效果 |
|---|---|
| 吃優格 | +1🟢 |
| 發酵食物（泡菜／味噌） | +1🟢 |
| 益生菌膠囊 | +2🟢（強力） |
| 多喝水 | −1🔴（最低留 1） |
| 高纖蔬果 | −1🔴（最低留 1） |
| 減糖斷捨離 | −1🔴（餓死壞菌，最低留 1） |
| 益生元套餐 | 本回合檢定 +1；**若手上有優格/發酵→合生元加乘再 +1🟢**（益生元餵養益生菌） |
| 規律運動 | 抵消一次「熬夜追劇」類懲罰 |
| 清淡飲食 | 獲得 🛡️清淡調養（下回合檢定 +1，前置調養） |
| 放鬆冥想 | 清除自己 1 個負面狀態（解連鎖） |
| 補充睡眠 | 立即多抽 1 張機會卡 |

### 6.5 事件連鎖關係（v1.2 新增）
把單回合事件串成「有前因後果」的生活敘事，強化教育訊息：
- **連鎖後續**：暴飲暴食 → 下回合必觸發「隔天腹瀉」（放縱的延遲代價）。
- **殘留狀態**：抗生素→菌相失衡、流感/壓力/便祕→發炎，中招者下回合更脆弱 → 呈現「壞事會拖累後續」。
- **正向連鎖**：益生元＋優格＝合生元加乘；清淡飲食＝為明天預先調養（前置 buff）。
- **解連鎖**：放鬆冥想清單一負面狀態、陽光好事件淨化全員 → 給玩家「主動止血」的選擇。

## 8. 護欄（動態平衡，非全滅）
- **🔴壞菌永遠 ≥ 1**：任何減壞菌效果最低留 1 張 → 教「共生」，不追求把壞菌清光。
- **菌叢區容量上限 10 張**（好菌＋壞菌合計）：滿 10 還要加卡時須先棄 1 張 → 逼取捨，避免無腦堆好菌，更像「維持平衡」。

## 9. 教育訊息（設計初衷）
> 面對相同的流感或美食誘惑，只有平時用機會卡維持好菌優勢的人，才能在共同考驗中健康勝出。
> **這不是運氣不好，而是平時菌叢底子不夠好。** 目標是維持動態平衡與好菌優勢，而非把壞菌徹底消滅。
